Advanced Programming in Python: Agenda

Venue: Room 1003, Building 58, Highfield Campus

The following agenda outlines what will be taught on the course. Additional notes on the teaching materials will be made available shortly after the course.

The software prerequisites for the lessons can be found here. The course training materials (which you can revisit and follow at your leisure) are linked to from the topics below.

The sessions run every monday from February 1 - March 21, 2016, 6:00 PM - 8:00 PM

.

Day 1: February 1

Python basics

In this session, you'll learn the fundamentals of Python. The following topics will be covered:
  • Basics - e.g. Starting Python, using variables
  • Working with arrays and lists
  • Basic program control - loops and conditionals
  • Creating and using functions
  • Using libraries

Day 2: February 8

Reading, analysing data and Using Libraries, Creating and Using functions.

In this session, you'll learn how to define and use Python functions and a basic scientific data analysis using Surface temperature data and patient inflammation data. The following topics will be covered:
  • Reading and analysing data
  • Using libraries
  • Creating and using functions
